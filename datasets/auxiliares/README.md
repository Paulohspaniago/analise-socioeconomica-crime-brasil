# auxiliares

Pasta para tabelas auxiliares usadas em cruzamentos e enriquecimento.

Uso recomendado:

- tabela de municipios do IBGE com `codigo_municipio` de 7 digitos
- equivalencias de nomes de municipios
- tabelas de apoio para padronizacao
- apenas capitais
- links : 
IDHM - http://www.atlasbrasil.org.br/consulta/planilha
populacao - https://basedosdados.org/dataset/1e2b9a88-9dc7-4f0e-a3a5-e8d2a13869bf?table=1a8d9636-c11d-443b-ae83-1b00576f0b70
crimes - https://forumseguranca.org.br/publicacoes/anuario-brasileiro-de-seguranca-publica/
educacao - https://qedu.org.br/brasil/baixar-dados?7&brasil
